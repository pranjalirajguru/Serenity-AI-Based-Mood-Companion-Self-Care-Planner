# 🌿 Serenity — AI-Based Mood Companion & Self-Care Planner

## 📌 Overview

Serenity is a UI/UX case study and interactive frontend prototype designed to help students and young professionals manage stress, build sustainable self-care habits, and improve emotional awareness.

This project combines **human-centered design principles** with a **functional UI prototype**, showcasing both UX thinking and frontend implementation.

---

## 🎯 Problem Statement

Students and young professionals often experience:

* High stress and burnout
* Lack of consistent self-care routines
* Poor awareness of emotional patterns

Existing wellness apps:

* Feel too clinical or overwhelming
* Lack personalization
* Create guilt through rigid routines

---

## 💡 Solution

Serenity provides a **calm, intelligent, and adaptive experience** that:

* Tracks daily mood patterns
* Suggests personalized self-care actions
* Encourages small, sustainable habits
* Reduces cognitive load through minimal UI

---

## 👤 Target Users

* Students (18–25)
* Young professionals
* Individuals experiencing stress, burnout, or anxiety

---

## ✨ Key Features

### 😊 Mood Tracking

* Quick daily check-ins
* Visual mood representation

### 🤖 AI-Based Suggestions

* Context-aware recommendations
* Gentle, non-intrusive guidance

### 📊 Progress Insights

* Weekly emotional trends
* Visual analytics for self-awareness

### ✅ Self-Care Tasks

* Small, achievable actions
* No pressure or guilt-based tracking

---

## 🎨 Design Principles

* **Calm & Minimal UI** → Reduces anxiety and cognitive overload
* **Human-Centered Design** → Focus on emotional experience
* **Low Friction** → Fast interactions (2–3 taps)
* **Non-Intrusive Experience** → No aggressive notifications

---

## 🧠 UX Decisions

* Soft pastel colors to create a calming environment
* Rounded components for emotional comfort
* Minimal text to reduce overwhelm
* Visual feedback instead of heavy instructions

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Custom design system)
* Responsive layout
* No external frameworks
